How to Generate Graph Files

1. Download graph-builder

Windows EXE. No Python, no pip needed.

Download graph-builder.exe

2. Run on your project

graph-builder.exe C:\path\to\your\project

Creates graphify-out/graph.json + .code-review-graph/graph.db

3. Upload or Clone

Either upload in the Upload tab, or add a Bitbucket project (+ New Project in sidebar).

MCP Server for Claude Code

Standalone (local)

Download and run locally. Graph stays on your machine.

Download mcp_server_standalone.py

Run: python mcp_server_standalone.py --crg-db .code-review-graph/graph.db --graphify graphify-out/graph.json

Online (pod)

Upload your graph + download .mcp.json with your private token. Place in project root.
